# 스터디 방식

1주 1회 온라인 모임

1인당 1시간씩  2명 발표

책내용 \+ 추가주제

책내용은 모두 공부해와서 스터디 당일에 발표할 사람 정함  
발표자료는 각자 준비  
발표자로 뽑혓는데 준비가 안되있으면 그 다음주 발표자로  고정함

추가주제는 자유주제.  
책 내용과 관련된 최근 트랜드  
그 주에 핫한 주제에 대한 발표  
수요일까지 발표자 모집  
추가주제 발표자는 그 주의 책내용 발표자 사다리타기에서 면제입니다

스터디 참여자 모두 1번 이상 책내용 발표,  1번 이상  추가주제 발표를 해야함
