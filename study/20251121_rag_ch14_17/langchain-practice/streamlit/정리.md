# streamlit

https://docs.streamlit.io/develop/api-reference

파이썬을 사용하여 데이터 애플리케이션을 빠르고 쉽게 만들 수 있도록 도와주는 오픈 소스 프레임워크
데이터 과학자와 머신러닝 엔지니어가 복잡한 웹 개발 기술 없이도 대화형 웹 애플리케이션을 구축할 수 있도록 설계

1. 간단한 문법
2. 실시간 업데이트
3. 위젯 지원 : 슬라이더, 버튼, 셀렉트 박스 등
4. 데이터 시각화 : Matplotlib, Plotly, Altair 등과 같은 파이썬 라이브러리와 쉽게 통합
5. 배포 용이성 : 로컬 서버에서 실행. 클라우드 서비스에 쉽게 배포 가능
6. 커뮤니티와 확장성 : 다양한 플러그인과 확장 기능을 통해 기능 확장

-   데이터 분석 결과 공유
-   머신러닝 모델 시연
-   대화형 데이터 대시보드

# streamlit component

## 텍스트 컴포넌트:

-   `st.text()`: 간단한 텍스트를 표시합니다.
-   `st.markdown()`: 마크다운 형식의 텍스트를 표시합니다.
-   `st.latex()`: LaTeX 수식을 렌더링합니다.
-   `st.code()`: 코드 블록을 표시합니다.

## 데이터 디스플레이 컴포넌트:

-   `st.dataframe()`: 데이터프레임을 대화형 테이블로 표시합니다.
-   `st.table()`: 정적 테이블을 표시합니다.
-   `st.json()`: JSON 데이터를 포맷팅하여 표시합니다.

## 차트 컴포넌트:

-   `st.line_chart()`: 선 그래프를 그립니다.
-   `st.area_chart()`: 영역 그래프를 그립니다.
-   `st.bar_chart()`: 막대 그래프를 그립니다.
-   `st.pyplot()`: Matplotlib 차트를 표시합니다.
-   `st.altair_chart()`: Altair 차트를 표시합니다.
-   `st.vega_lite_chart()`: Vega-Lite 차트를 표시합니다.
-   `st.plotly_chart()`: Plotly 차트를 표시합니다.
-   `st.bokeh_chart()`: Bokeh 차트를 표시합니다.

## 위젯 컴포넌트:

-   `st.button()`: 버튼을 생성합니다.
-   `st.checkbox()`: 체크박스를 생성합니다.
-   `st.radio()`: 라디오 버튼을 생성합니다.
-   `st.selectbox()`: 드롭다운 선택 상자를 생성합니다.
-   `st.multiselect()`: 다중 선택 상자를 생성합니다.
-   `st.slider()`: 슬라이더를 생성합니다.
-   `st.select_slider()`: 선택 슬라이더를 생성합니다.
-   `st.text_input()`: 텍스트 입력 상자를 생성합니다.
-   `st.text_area()`: 텍스트 영역을 생성합니다.
-   `st.number_input()`: 숫자 입력 상자를 생성합니다.
-   `st.date_input()`: 날짜 입력 위젯을 생성합니다.
-   `st.time_input()`: 시간 입력 위젯을 생성합니다.
-   `st.file_uploader()`: 파일 업로드 위젯을 생성합니다.
-   `st.color_picker()`: 색상 선택기를 생성합니다.

## 미디어 컴포넌트:

-   `st.image()`: 이미지를 표시합니다.
-   `st.audio()`: 오디오 파일을 재생합니다.
-   `st.video()`: 비디오 파일을 재생합니다.

## 레이아웃 및 컨테이너 컴포넌트:

-   `st.sidebar`: 사이드바를 생성하여 위젯을 배치할 수 있습니다.
-   `st.container()`: 컨테이너를 생성하여 여러 요소를 그룹화할 수 있습니다.
-   `st.columns()`: 여러 열을 생성하여 레이아웃을 구성할 수 있습니다.
-   `st.expander()`: 확장 가능한 컨테이너를 생성합니다.

# `with` 문법

**컨텍스트 매니저(context manager)** 열고, 쓰고, 자동으로 정리.

-   파일 자동으로 닫기
-   락(lock) 자동 해제
-   데이터베이스 연결/트랜잭션 정리
-   리소스(메모리, 세션 등) 정리

```python
with 표현식 as 변수:
	# 이 블록 안에서 변수 사용
	...
# 블록을 빠져 나오면 자동 정리
```

```python
with open("example.txt", "r", encoding="utf-8") as f:
	content = f.read()
	print(content)
# 여기까지 오면 f.close()가 자동으로 호출됨. 예외 발생에도 자동 호출
```

`with` 가 없으면?

```python
f = open("example.txt", "r", encoding="utf-8")
try:
	content = f.read()
finally:
	f.close()
```
